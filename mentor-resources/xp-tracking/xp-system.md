# XP & Badge System

Total program XP: **3,400**.

## How to earn XP

| Action | XP |
| ------ | ---: |
| Attend a session | 25 |
| Submit a project on time | 50 |
| Submit a capstone | 150 |
| Capstone scoring 90+ | +50 bonus |
| Open a thoughtful Discussion / Issue | 15 |
| Review another mentee's PR | 25 |
| First merged PR (program lifetime) | 100 bonus |
| Help a peer in office hours | 20 |
