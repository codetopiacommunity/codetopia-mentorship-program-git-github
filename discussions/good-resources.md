# Good Resources (curated)

A short, opinionated list. Bookmark these.

## Official docs

- [Pro Git book (free)](https://git-scm.com/book/en/v2) — the canonical reference.
- [GitHub Docs](https://docs.github.com) — for everything GitHub-specific.

## Interactive practice

- [Learn Git Branching](https://learngitbranching.js.org/) — visual, addictive.
- [Oh My Git!](https://ohmygit.org/) — Git as a card game.
- [Git Explorer](https://gitexplorer.com/) — "I want to ___ in Git" → command.

## Video

- **FreeCodeCamp** — full-length Git & GitHub course on YouTube.
- **Net Ninja** — short, well-edited Git playlist.
- **Fireship** — "Git Explained in 100 Seconds" and longer rants.

## Reading

- ["Think Like (a) Git"](https://think-like-a-git.net/) — for after you've felt the pain.
- ["A Hacker's Guide to Git"](https://wildlyinaccurate.com/a-hackers-guide-to-git/) — the model under the model.
- [Conventional Commits](https://www.conventionalcommits.org/) — the spec we use.

## Cheatsheets

- [GitHub's Git Cheatsheet (PDF)](https://education.github.com/git-cheat-sheet-education.pdf)
- Our own: [`../resources/git-cheatsheet.md`](../resources/git-cheatsheet.md)
