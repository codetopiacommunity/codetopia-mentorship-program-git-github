# Projects Showcase

The trophy case. Mentees publish their capstone work here.

## How to add yours

1. Fork the repo.
2. Create a folder: `projects-showcase/week-<n>-projects/<your-handle>/`.
3. Add at minimum:
   - `README.md` (use [`../templates/readme-template.md`](../templates/readme-template.md))
   - Your project files
   - A screenshot
4. Open a PR; tag your mentor.

## Folder layout

```
projects-showcase/
├── week-1-projects/
│   └── ada-lovelace/
│       ├── README.md
│       └── ...
├── week-2-projects/
└── ...
```
