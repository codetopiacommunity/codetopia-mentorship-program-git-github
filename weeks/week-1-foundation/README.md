## Big Picture

Git is a tool that records changes to files. That's it. By the end of this week, mentees should have:

1. A working Git installation with their identity configured.
2. A local repository they created from scratch.
3. At least 5 atomic commits with conventional messages.
4. A pushed copy on GitHub.
