# Week 1 Capstone Rubric

Total: 100. Pass: 70+. Stretch adds up to +50 XP.

| Criterion | Points | Great looks like |
| --------- | -----: | ---------------- |
| Repo set up correctly | 10 | `.git/` exists, `main` is the default branch |
| README clear and inviting | 15 | Title, description, how-to-read, your name |
| At least 8 atomic commits | 20 | One logical change per commit |
| Conventional commit messages | 15 | All start with a valid type, imperative |
| `.gitignore` correct | 10 | No `.DS_Store`, no `node_modules/`, no `.env` |
| Content quality | 15 | Real content — not placeholder lorem ipsum |
| "What I learned" section | 10 | Specific, honest, names a stuck moment |
| File organization | 5 | Sensible structure, no junk files |
| **Total** | **100** | |

## Stretch (+50)

| Item | XP |
| ---- | -: |
| Pushed to your own GitHub | +20 |
| At least one `fix:` commit | +10 |
| Styled with CSS | +20 |
