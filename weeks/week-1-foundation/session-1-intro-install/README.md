# Week 1 · Session 1 · Introduction & Installation

> **XP:** 75
> **Time:** 60–90 min
> **Prereqs:** A computer and a terminal you've opened at least once.

## 🎯 Learning Objectives

By the end of this session you will:

1. Explain what version control is and *why* it exists.
2. Have Git installed on your machine.
3. Have configured `user.name`, `user.email`, and `init.defaultBranch`.
4. Run `git --version` and `git config --list` successfully.
