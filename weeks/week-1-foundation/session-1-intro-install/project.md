# Project — Session 1

> **XP:** 50

## What you'll do

Get Git installed and working on your machine, then prove it.

## Steps

1. Install Git for your OS (see lecture notes).
2. Configure your identity (`user.name`, `user.email`, `init.defaultBranch`).
3. Run `git --version` and `git config --list`.
4. Create a folder called `hello-git`, run `git init` inside it.
5. Run `ls -la` and confirm `.git/` exists.

## Deliverable

In the **Discussions** tab, post a single message containing:

- The output of `git --version`
- The output of `git config --get user.name` and `git config --get user.email`
- A one-sentence answer to: *"What is the difference between Git and GitHub?"*
