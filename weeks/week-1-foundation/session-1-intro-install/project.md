# Project — Session 1

> **XP:** 50

## What you'll do

Get Git installed and working on your machine, then prove it.

## Steps

1. Install Git for your OS (see lecture notes).
2. Configure your identity (`user.name`, `user.email`, `init.defaultBranch`).
3. Run `git --version` and `git config --list`.
4. Create a folder called `hello-git`, run `git init` inside it.
5. Run `ls -la` and confirm `.git/` exists.

## Deliverable

Submit your work as follows:

1. **In your project's `screenshots/` folder** (the submission of record): add at least one screenshot — typically a terminal shot of `git --version` plus `git config --get user.name` and `git config --get user.email`. Name them `01-git-version.png`, `02-git-config.png`, etc.
