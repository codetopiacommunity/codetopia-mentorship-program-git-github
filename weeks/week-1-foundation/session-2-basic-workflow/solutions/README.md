# Solutions — Session 2

> **Mentor only.**

## Example clean log

```
e7a1c0d feat: add credits footer
b1f4592 feat: add favorite quotes section
a09c321 feat: add favorite music section
78b2f10 feat: add favorite movies section
3d12cab feat: scaffold favorites README
```

## Common feedback to give

- "Your messages describe *what* — try one that describes *why*."
- "Two changes in one commit — split into two next time."
- "Great atomic commits!"

## Misuses to flag

- Single giant commit ("initial commit" with everything) — coach toward atomicity.
- `git add .` reflexively — model `git add <file>` first, then graduate.
- Past-tense messages ("added books") — gently correct to imperative.
