# Week 1 · Session 2 · The Basic Git Workflow

> **XP:** 75
> **Time:** 60–90 min
> **Prereqs:** Session 1 complete.

## 🎯 Learning Objectives

By the end of this session you will:

1. Initialize a repo and explain what `git init` actually did.
2. Use `git add`, `git commit`, and `git status` fluently.
3. Visualize the **3 Git states** (working / staging / repo).
4. Write a clear conventional commit message.
