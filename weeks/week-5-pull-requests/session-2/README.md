# Week 5 · Session 2 · Code Review — Comments, Changes, Merges

> **XP:** 75
> **Time:** 60–90 min
> **Prereqs:** Session 1. You have opened at least one PR.

## Learning Objectives

By the end of this session you will:

1. Leave review comments that are kind, specific, and actionable.
2. Use the **suggested changes** feature to propose exact diffs.
3. Choose between **Comment**, **Approve**, and **Request changes** on purpose.
4. Respond to feedback by pushing additional commits (not force-pushing in most cases).
5. Pick a merge strategy — **merge commit**, **squash**, or **rebase** — with a reason.
